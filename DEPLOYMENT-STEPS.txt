---

# ✅ STEP 3 — Backend Deployment (Express Serverless)

### ⭐ Backend Folder

```
ecommerce-server
```

---

## ⭐ Express must be serverless

### ✔ api/index.js

```js
import express from "express";
const app = express();

/* routes */

export default app;
```

❌ DO NOT USE:

```js
app.listen(5000)
```

---

## ⭐ Backend vercel.json

```
ecommerce-server/vercel.json
```

```json
{
  "routes": [
    {
      "src": "/(.*)",
      "dest": "/api/index.js"
    }
  ]
}
```

---

## ⭐ Add ENV variables in Vercel Dashboard

Go:

```
Vercel → ecommerce-server → Settings → Environment Variables
```

Add:

```
RAZORPAY_KEY_ID
RAZORPAY_KEY_SECRET
DATABASE_URL
```

---

## ⭐ Deploy Backend

```bash
cd ecommerce-server
vercel
vercel --prod
```

You will get URL:

```
https://ecommerce-server-five-eosin.vercel.app
```

👉 This is your **LIVE API URL**

# ✅ STEP 4 — Configure Angular Environment

### ⭐ Local

```
src/environments/environment.ts
```

```ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:5000',
  razorpayKey: 'rzp_test_7loOYJatPLp2HD'
};
```

---

### ⭐ Production

```
src/environments/environment.prod.ts
```

```ts
export const environment = {
  production: true,
  apiUrl: 'https://ecommerce-server-five-eosin.vercel.app',
  razorpayKey: 'rzp_test_7loOYJatPLp2HD'
};
```

---

# ✅ STEP 5 — Angular Production Replacement

Already correct in `angular.json`

```json
"fileReplacements": [
  {
    "replace": "src/environments/environment.ts",
    "with": "src/environments/environment.prod.ts"
  }
]
```

👉 When deployed → Angular uses LIVE API automatically

---

# ✅ STEP 6 — Use Environment in Code

Example:

```ts
this.http.get(environment.apiUrl + '/orders')
```

Example Razorpay:

```ts
key: environment.razorpayKey
```
